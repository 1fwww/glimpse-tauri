Chat Header — Layout Options
A. Eye as thread menu trigger The eye icon replaces the chevron as the thread menu trigger. Clicking the eye or the title opens thread list. New chat gets its own button. Eye stays as brand + functional anchor.
brand + menu trigger new chat (branded accent) pin + close (dim, secondary)
B. Eye anchored left, clear groups right Eye is purely brand (decorative + wiggle trigger). Title + chevron is the thread menu. New chat, pin, close grouped right with a visual separator between new chat and window controls.
brand (decorative + wiggle) title = thread menu new chat | separator | pin + close
C. Eye + title fused, compact action group Eye and title form a single clickable unit (thread menu). No chevron — the eye itself signals "clickable." Actions are compact: new chat uses a compose icon (pen) instead of (+) to differentiate from "add" semantics.
eye + title = fused menu trigger compose (new chat) pin + close
